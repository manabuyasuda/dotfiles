# Sections

このファイルはマークアップルールの全セクション・優先度・概要を定義します。セクションID（括弧内）はファイル名のprefixとして使用します。

## 1. Semantic (semantic)

ImpactはHIGHです。要素の役割（リンク・ボタン・リスト等）からタグを選定します。a11yツリーで「何ができる要素か」が正しく伝わることを目指します。

## 2. Accessibility (a11y)

ImpactはHIGHです。キーボード操作・フォーカス可視性など、マウス以外の操作手段を保ちます。

## 3. Performance (perf)

ImpactはHIGHです。画像の解像度最適化など、表示パフォーマンスに影響するマークアップを扱います。
